Temperature units are displayed in Celcius.
Humidity units are displayed in Relative Humidity %, or current absolute humidity relative to the maximum for the current temperature.


Specifications:
	Wide operating voltage range:
		(3.3V ~ 5V)

	Low Power Consumption:
		350 µA during RH conversion

	0 to 100% RH operating range

	Measuring Range:
		Humidity: 0% - 80% RH
		Temperature: 0 ~ 70 °C

	Accuracy:
		Humidity: ±4.5% RH
		Temperature: ±0.5°C

	I2C host interface
	Excellent long term stability
